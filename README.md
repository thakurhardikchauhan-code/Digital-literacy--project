# Digital-Literacy-Project
This report gives a brief overview of a Digital Literacy project covering practical tasks such as creating an infographic, building a digital portfolio, exploring coding and collaboration platforms, learning email etiquette, and understanding cybercrime awareness.
Digital Literacy Project
Name: Sameeksha Mishra
Course: CSE0001 – Digital Literacy

Project Overview
This project is part of my Digital Literacy course. Through these tasks, I got to explore different digital tools and understand how to use them properly in real life, especially as a student.

Task Summary
Task 1 – Infographic
I made an infographic using Canva to explain digital literacy in a simple way. I included topics like safe internet use, useful tools, and basic email etiquette.

Task 2 – Digital Portfolio
In this task, I created my profiles on platforms like GitHub, LinkedIn, Kaggle and Stack Overflow. This helped me understand how important it is to have an online presence.

Task 3 – Platforms
I solved a beginner-level coding problem on HackerRank, which was my first coding experience on such a platform. I also created a Google Form quiz on digital literacy.

Task 4 – Email Etiquette
I wrote two professional emails and made a checklist of do’s and don’ts for social media use.

Task 5 – Cybercrime Awareness
I worked on a case study about UPI fraud and made a checklist on how to stay safe online.
